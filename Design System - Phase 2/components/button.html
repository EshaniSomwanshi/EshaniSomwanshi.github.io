Accessibility Considerations

Ensure accessibility by using role="button" on anchor tags to clearly communicate their purpose to assistive technologies. Added aria-label attributes to describe each buttonâ€™s function, especially for screen-reader users. Use of aria-disabled="true" and tabindex="-1" for inactive buttons to prevent keyboard interaction. Hid decorative icons from screen readers with aria-hidden="true" to avoid unnecessary distractions.
